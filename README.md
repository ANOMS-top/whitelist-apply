# Whitelist Apply
ANOMS服务器白名单申请网站！

使用Python&HTML编写！

如何启动：
```bash
pip install flask                                           # 安装Flask
git clone https://github.com/ANOMS-top/whitelist-apply.git  # 克隆本项目
cd whitelist-apply                                          # cd到本项目目录
python main.py                                              # 启动项目
```

包含了一些[Redroadsl/Minecraft-Rcon](https://github.com/Redroadsl/Minecraft-Rcon)（MIT）的代码
